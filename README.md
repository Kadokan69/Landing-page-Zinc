# Geberich - Цинк + Селен для волос и кожи

Лендинг для витаминного комплекса **Цинк и Селен** от бренда Geberich. Сайт разработан для презентации продукта, его преимуществ, механизмов действия и призыва к покупке на маркетплейсах.

## О проекте

Одностраничный сайт, созданный для информирования потенциальных покупателей о пользе витаминов цинка и селена для здоровья волос, кожи и ногтей. Страница включает структурированное описание продукта, его свойства, рекомендации по применению и ссылки на популярные маркетплейсы для покупки.

## Возможности

- Адаптивная верстка
- Бургер-меню для мобильной версии
- Всплывающее окно с подробным составом продукта
- SEO-оптимизированная структура

## Технологии

- HTML5 с семантической версткой
- CSS3
- JavaScript (ES6 модули)
- JSON-LD

## Структура страницы

- Шапка с логотипом и навигацией
- Главный экран с продвижением продукта и ссылками на маркетплейсы
- Блок с подробным описанием действия комплекса
- Информация о пользе для иммунитета и кожи
- Механизмы действия (3 ключевых пункта)
- Рекомендации по применению и противопоказания
- Подвал с юридической информацией
- Всплывающее окно с составом

## SEO-оптимизация

- Корректный заголовок и мета-описание
- Open Graph теги для соцсетей
- Правильная иерархия заголовков (h1-h3)
- Осмысленные alt-тексты для изображений
- Микроразметка Product в формате JSON-LD

## Поддержка браузеров

Chrome, Firefox, Safari, Edge, Opera (последние версии)
